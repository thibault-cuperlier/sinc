Pour utiliser ce chiffrement RSA fait maison, suivez ces étapes simples :

1. Générez deux grands nombres premiers p et q en exécutant le script `aleatoir.py` :
python aleatoir.py
2. Ensuite, démarrez le script `chifrement.py` avec :
python chifrement.py
3. Entrez le nombre p généré par le premier script, appuyez sur Entrée, et faites la même chose pour q.
4. Enfin, démarrez le cryptage du message.
