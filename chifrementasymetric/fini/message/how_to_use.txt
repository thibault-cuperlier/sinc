# Guide d'utilisation du chiffrement RSA

Ce guide explique comment utiliser les scripts pour générer des clés RSA, chiffrer un message et le déchiffrer.

---

## Étape 1 : Générer deux grands nombres premiers

Exécutez le script `aleatoir.py` pour générer deux grands nombres premiers `p` et `q` :

```bash
python [aleatoir.py](http://_vscodecontentref_/0)
```
